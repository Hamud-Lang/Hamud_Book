# 目录文件结构释义

```bash
/assets             {附录资源目录}

/book               {编译后文件存放目录}

/theme              {主题外观相关文件存放目录}

/book.toml          {总配置文件}

/src                {文章内容文件存放目录}
├─……                {其它目录或文章}
├─SUMMARY.md        {文章总目录}
├─index.md          {主内容文章}
└─Contribute        {代码或内容贡献相关内容存放目录}
    ├─index.md      {主内容文章}
    ├─File_structure.md
    ├─Creation.md   {内容创作相关文章}
    └─……            {其他目录或文章}
```

> 更详细的内容请查看 [mdbook 文档](https://rust-lang.github.io/mdBook/)。
