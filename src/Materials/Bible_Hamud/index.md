# 哈约·姆经（中文版）

> 哈姆族经典《`hamud`》（中文译作《哈姆德》）讲述了哈姆人的历史与宗教，是研究哈姆学必不可少的一部传世之作。本译版将“`hamud`”译作“`哈姆`”，部分地方为尊重原作未做翻译。
